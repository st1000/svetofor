**Дело:** А36-6051/2026  

## Что такое «обмен с 1С» по договору (важно)

**По ТЗ и договору обмен с 1С — это API исполнителя на домене `api.svetofor-mebel.ru`**, а не старый бекенд Ильи Кулькина (до договора).

| | Обмен по договору (этап 2) | Старый бекенд Ильи (вне договора) |
|---|---------------------------|-----------------------------------|
| **Где** | `http://api.svetofor-mebel.ru/api/v1/integration/1c/` | отдельный проект Ильи («бекенд Ильи») |
| **Документация** | `http://api.svetofor-mebel.ru/api/docs#/integration-1c/` | переписка до договора |
| **Кто делал** | ИП Ляпин (этап 2 ТЗ) | веб-студия заказчика до договора |
| **На скринах** | ссылки Кирилла, JSON `products`, `external_id` | «Илья обнулит базу», «на бекенд ильи» |

**Доказательства, что обмен по договору существовал и работал:**
- `скриншоты/Снимок экрана 2026-06-25 в 15.05.23.png` — в чате ссылка на **Swagger** `integration-1c`, Кирилл: «со стороны сайта ниче не меняется»
- `скриншоты/Снимок экрана 2026-06-25 в 15.07.42.png` — `.../integration/1c/products`, Кирилл: товары «**напрямую выгружены с 1с**»
- `скриншоты/Снимок экрана 2026-06-25 в 15.01.24.png`, `15.07.28.png` — те же эндпоинты
- MAX `photo_2026-06-25 18.18.55.jpeg`–`18.19.00.jpeg` — тесты через `api.svetofor-mebel.ru/.../integration/1c/`

Заказчик в переписке **путает** два контура (свой бекенд Ильи vs `api.svetofor`), но **оплатил этап 2** и дальше **сам** крутил выгрузки/ID/1С (Роман, Илья) — это не «обмен не сделан», а доработка и отладка **на их стороне** (ст. 406, 718 ГК).

**Тезис для суда:** обмен с 1С по договору = **`api.svetofor-mebel.ru`**; оплата этапа 2 — **04.02.2026** (ПП № 34). После оплаты заказчик месяцами менял выгрузки, ID, структуру номенклатуры в 1С и параллельно тянул старый бекенд Ильи — **просрочка кредитора**, не невыполнение этапа 2.

**Ориентиры:**
- Оплата 2-го этапа: **04.02.2026**
- Срок по п. 2.1 договора (позиция истца): **~18.03.2026**

## Формулировка для суда

> Обмен с 1С по договору (этап 2) — API на **`http://api.svetofor-mebel.ru/api/v1/integration/1c/`** (документация: `.../api/docs#/integration-1c/`). Заказчик **оплатил этап 2 04.02.2026**; в переписке есть ссылки на эндпоинты, JSON товаров с `external_id`, формулировка «**напрямую выгружены с 1с**». После оплаты представители заказчика (**Роман**, **Илья**, **Костяхин**) **многократно меняли** выгрузки и ID на стороне 1С и параллельно тянули старый бекенд Ильи — не путать с `api.svetofor`. В мае 2026: «**обмен дрочит со стороны 1с**», «**полдня тестили обмен**». Претензия об «невыполненном обмене» **противоречит** оплате этапа 2 и наличию рабочего API (ст. 406, 718, 401 ГК РФ).
